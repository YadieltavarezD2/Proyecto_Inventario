# Proyecto-invntario
# Descripcion:
El porgrama permite registrar movimintos de entrada y salida de productos hacia clientes o proveedores , ademas de poder ver, generar y estionar los reportes hechos por el mismo , tambien cuenta con distintos niveles de acceso en este caso admin, supervisor y almacen.
# Herramientas utilizadas:

Lenguaje: C#

Framework: .NET Framework

IDE: Visual Studio

Gestor de base de datos: SQL Server 20.2.1

Arquitectura: Proyecto en capas (Presentación, Negocio, entidades, Datos)

Tecnología de acceso a datos: ADO.NET

Reportes: RDLC (Report Definition Language Client-side)

Control de visualización: Microsoft ReportViewer

# Uso o Ejecución:

Clonar el repositorio desde GitHub

bash:

```bash
git clone https://github.com/YadieltavarezD2/Proyecto-Proecto_Inventario
```

Abrir el archivo .sln en Visual Studio

Compilar el proyecto

Ejecutar la aplicación presionando F5 o el botón Iniciar

La aplicación se abrirá mostrando el formulario principal
